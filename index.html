What I've Built

🔥
GuardianEye
AI-Powered Surveillance System

GuardianEye is an AI-powered real-time surveillance system designed to detect fire hazards and road accidents from video streams and trigger immediate alerts, helping reduce response time and potential damage in critical situations.

The system processes both live and recorded video feeds including CCTV cameras, drone streams, and uploaded media. Using computer vision and deep learning models, it identifies fire, smoke, and accident events and further classifies incidents based on severity (minor vs critical).

The platform is built as a full-stack web application with a responsive monitoring dashboard. The frontend is deployed on Vercel, the backend runs on Render handling APIs and alert systems, and MongoDB Atlas is used for storing logs, alerts, and metadata.

Due to high computational requirements, the AI model runs locally and communicates with the backend via REST APIs for real-time inference, ensuring efficient and scalable processing.

Key Impact: This system enables early detection of dangerous events, reduces emergency response time, and can be extended for applications in smart cities, industrial safety, and disaster management systems.
